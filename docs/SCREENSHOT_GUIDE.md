# Screenshot Guide for Devpost

To ensure a high-quality Devpost submission, capture the following screenshots in full screen (1920x1080) in dark mode.

1. **Dashboard_Overview.png**
   - URL: `/`
   - Capture the entire dashboard immediately after the stagger animation completes.

2. **Digital_Twin_3D.png**
   - URL: `/digital-twin`
   - Orbit the camera so you can see a busy intersection with cars stopped at a red light. 
   - Ensure the UI sidebar is visible.

3. **Crisis_Simulation_Timeline.png**
   - URL: `/simulation`
   - Run the Demo mode for the *Major Traffic Accident*.
   - Pause the simulation (Press `Space` bar) halfway through when the timeline says "Recommendation Generated".
   - Capture the screen showing the red Critical toast notification in the corner.

4. **IoT_Telemetry_Center.png**
   - URL: `/telemetry`
   - Wait for the Emergency Feed to populate with at least two items.
   - Capture the screen showing the live Camera feeds.

5. **AI_Studio_Pipeline.png**
   - URL: `/ai-studio`
   - Capture the neural pipeline visualizer when it is actively analyzing a threat.

6. **Global_Command_Palette.png**
   - URL: Any page
   - Press `Ctrl+K`. Type "telemetry".
   - Capture the screen with the blurred background and the active command palette.
