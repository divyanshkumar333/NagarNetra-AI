# NagarNetra AI (नगरनेत्र AI)
> The Intelligent Eye of the City.

NagarNetra AI is an AI-powered Smart City Operations Command Center focused on traffic intelligence, urban monitoring, predictive analytics, and emergency response.

## Overview
This platform provides national traffic control centers with real-time digital twins, advanced AI simulation, and operations management.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS, shadcn/ui, Framer Motion, React Three Fiber.
- **Backend**: Python, FastAPI.

## Getting Started
Run the `start.bat` script to initialize the virtual environments, install dependencies, and launch both frontend and backend servers.
See `docs/DeveloperSetup.md` for more details.
